# Wonder Workshop Coding Challenge

# Instruction

- Clone this repo to your local machine
- Navigate to where package.json is located
- Open up a shell and 
```sh
npm install
```
```sh
npm run build
```
- Navigate to the 'static' folder
- Double click on index.html
- The 'news' subreddit is the default and you can add more subreddit by searching
- If the search subreddit doesn't exists nothing will come up
- You can only delete a subreddit if there is more than one
- Scroll down to see more posts, it might take a few seconds for it to retreive the new posts
